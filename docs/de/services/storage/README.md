# Storage Services

Storage Services für File Storage und Synchronisation.

## Services

- [OwnCloud](./owncloud.md) - File Storage

## Übersicht

OwnCloud ist eine Open-Source File Storage Lösung für deine Dateien mit Sync-Funktionalität.

## Features

- File Storage & Sync
- Web Interface
- Desktop Client
- Mobile Apps
- File Sharing
- Version Control

## Setup

1. OwnCloud deployen
2. Admin Account erstellen
3. Desktop Client oder Mobile App installieren
4. Verbinde mit Server

## Datenbank

OwnCloud nutzt MySQL als Backend. Die Datenbank wird automatisch mit deployed.
