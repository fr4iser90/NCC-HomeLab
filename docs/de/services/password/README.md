# Password Management Services

Password Management Services für sichere Passwort-Verwaltung.

## Services

- [Bitwarden](./bitwarden.md) - Password Manager (Vaultwarden)

## Übersicht

Vaultwarden ist eine kompatible Implementierung des Bitwarden Servers für Self-Hosting.

## Features

- Passwort-Synchronisation
- Zwei-Faktor-Authentifizierung
- Secure Notes
- Browser Extensions
- Mobile Apps

## Setup

1. Bitwarden deployen
2. Web Vault öffnen
3. Account erstellen
4. Browser Extension oder Mobile App installieren

## Sicherheit

- Verschlüsselung auf Client-Seite
- Admin Panel nur über VPN/LAN
- Rate Limiting aktiviert
