# Companion Services

Companion Services erweitern die Funktionalität anderer Services.

## Services

- [Cloudflare Companion](./cloudflare.md) - DNS Management für Traefik

## Übersicht

Cloudflare Companion verwaltet automatisch DNS-Einträge in Cloudflare für Traefik Services.

## Setup

1. Cloudflare API Token erstellen
2. Companion konfigurieren
3. Companion starten
4. DNS-Einträge werden automatisch erstellt

## Abhängigkeiten

- Traefik muss laufen
- Cloudflare Account mit API Token
