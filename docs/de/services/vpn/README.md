# VPN Services

VPN Services für sicheren Remote-Zugriff auf dein Homelab.

## Services

- [WireGuard](./wireguard.md) - VPN Server

## Übersicht

WireGuard ist ein moderner, schneller VPN-Server für sicheren Remote-Zugriff.

## Features

- Moderne VPN-Technologie
- Schnelle Verbindungen
- Niedrige Latenz
- Web UI (optional)
- QR-Code für Mobile Setup

## Setup

1. WireGuard deployen
2. Port Forwarding im Router (51820/UDP)
3. Clients erstellen
4. Verbindung testen

## Wichtig

- Port 51820/UDP muss im Router forwardiert werden
- Web UI sollte nur über VPN/LAN erreichbar sein (empfohlen)
