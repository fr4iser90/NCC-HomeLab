# Media Services

Media Services für Streaming und Medienverwaltung.

## Services

- [Jellyfin](./jellyfin.md) - Open-Source Media Server
- [Plex](./plex.md) - Media Server

## Unterschiede

### Jellyfin
- ✅ Open Source
- ✅ Kostenlos
- ✅ Alle Features frei
- ⚠️ Weniger Client Apps

### Plex
- ✅ Viele Client Apps
- ✅ Einfacheres Setup
- ⚠️ Premium Features kostenpflichtig
- ⚠️ Proprietär

## Empfehlung

- **Jellyfin:** Für Open-Source Fans
- **Plex:** Für einfacheres Setup und mehr Clients
