# Games Services

Games Services für Game Server Management und Hosting.

## Services

- [PufferPanel](./pufferpanel.md) - Game Server Management

## Übersicht

PufferPanel ist ein Web-basiertes Game Server Management Panel für verschiedene Game Server.

## Features

- Game Server Management
- Web Interface
- File Manager
- Console Access
- Resource Monitoring

## Setup

1. PufferPanel deployen
2. Admin Account erstellen
3. Game Server hinzufügen
4. Ports konfigurieren

## Wichtig

- Game Ports müssen im Router forwardiert werden
- TCP Router in Traefik für Daemon nötig
