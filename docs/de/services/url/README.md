# URL Management Services

URL Management Services für URL Shortening und Link Management.

## Services

- [Yourls](./yourls.md) - URL Shortener

## Übersicht

Yourls (Your Own URL Shortener) ist ein Self-Hosted URL Shortener für deine eigenen kurzen Links.

## Features

- URL Shortening
- Custom Short Links
- Link Statistics
- QR-Code Generation
- API Support

## Setup

1. Yourls deployen
2. Admin Account erstellen
3. API Keys konfigurieren (optional)
4. Beginne Links zu kürzen

## Datenbank

Yourls nutzt MySQL als Backend. Die Datenbank wird automatisch mit deployed.
