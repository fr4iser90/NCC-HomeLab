# Dashboard Services

Dashboard Services bieten eine zentrale Übersicht über alle Homelab Services.

## Services

- [Organizr](./organizr.md) - Unified Dashboard

## Übersicht

Organizr ist ein HTPC/Homelab Services Organizer mit einem schönen Dashboard für alle deine Services.

## Features

- Unified Dashboard
- Tabbed Interface
- Service Status Monitoring
- Bookmark Management

## Setup

1. Organizr deployen
2. Admin Account erstellen
3. Services/Tabs hinzufügen
4. Dashboard konfigurieren
