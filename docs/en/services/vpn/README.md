# VPN Services

VPN services for secure remote access to your Homelab.

## Services

- [WireGuard](./wireguard.md) - VPN Server

## Overview

WireGuard is a modern, fast VPN server for secure remote access.

## Features

- Modern VPN technology
- Fast connections
- Low latency
- Web UI (optional)
- QR-code for mobile setup

## Setup

1. Deploy WireGuard
2. Port forwarding in router (51820/UDP)
3. Create clients
4. Test connection

## Important

- Port 51820/UDP must be forwarded in router
- Web UI should only be accessible via VPN/LAN (recommended)
