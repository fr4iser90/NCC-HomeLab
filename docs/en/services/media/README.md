# Media Services

Media services for streaming and media management.

## Services

- [Jellyfin](./jellyfin.md) - Open-Source Media Server
- [Plex](./plex.md) - Media Server

## Differences

### Jellyfin
- ✅ Open Source
- ✅ Free
- ✅ All features free
- ⚠️ Fewer client apps

### Plex
- ✅ Many client apps
- ✅ Easier setup
- ⚠️ Premium features paid
- ⚠️ Proprietary

## Recommendation

- **Jellyfin:** For open-source fans
- **Plex:** For easier setup and more clients
