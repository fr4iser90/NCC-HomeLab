# Games Services

Games services for game server management and hosting.

## Services

- [PufferPanel](./pufferpanel.md) - Game Server Management

## Overview

PufferPanel is a web-based game server management panel for various game servers.

## Features

- Game server management
- Web interface
- File manager
- Console access
- Resource monitoring

## Setup

1. Deploy PufferPanel
2. Create admin account
3. Add game servers
4. Configure ports

## Important

- Game ports must be forwarded in router
- TCP router in Traefik needed for daemon
