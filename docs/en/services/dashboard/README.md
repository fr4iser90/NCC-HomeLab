# Dashboard Services

Dashboard services provide a central overview of all Homelab services.

## Services

- [Organizr](./organizr.md) - Unified Dashboard

## Overview

Organizr is an HTPC/Homelab services organizer with a beautiful dashboard for all your services.

## Features

- Unified dashboard
- Tabbed interface
- Service status monitoring
- Bookmark management

## Setup

1. Deploy Organizr
2. Create admin account
3. Add services/tabs
4. Configure dashboard
