# Storage Services

Storage services for file storage and synchronization.

## Services

- [OwnCloud](./owncloud.md) - File Storage

## Overview

OwnCloud is an open-source file storage solution for your files with sync functionality.

## Features

- File storage & sync
- Web interface
- Desktop client
- Mobile apps
- File sharing
- Version control

## Setup

1. Deploy OwnCloud
2. Create admin account
3. Install desktop client or mobile app
4. Connect to server

## Database

OwnCloud uses MySQL as backend. The database is automatically deployed with it.
