# URL Management Services

URL management services for URL shortening and link management.

## Services

- [Yourls](./yourls.md) - URL Shortener

## Overview

Yourls (Your Own URL Shortener) is a self-hosted URL shortener for your own short links.

## Features

- URL shortening
- Custom short links
- Link statistics
- QR-code generation
- API support

## Setup

1. Deploy Yourls
2. Create admin account
3. Configure API keys (optional)
4. Start shortening links

## Database

Yourls uses MySQL as backend. The database is automatically deployed with it.
