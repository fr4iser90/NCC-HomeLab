# Password Management Services

Password management services for secure password management.

## Services

- [Bitwarden](./bitwarden.md) - Password Manager (Vaultwarden)

## Overview

Vaultwarden is a compatible implementation of the Bitwarden server for self-hosting.

## Features

- Password synchronization
- Two-factor authentication
- Secure notes
- Browser extensions
- Mobile apps

## Setup

1. Deploy Bitwarden
2. Open web vault
3. Create account
4. Install browser extension or mobile app

## Security

- Client-side encryption
- Admin panel only via VPN/LAN
- Rate limiting enabled
