# Companion Services

Companion services extend the functionality of other services.

## Services

- [Cloudflare Companion](./cloudflare.md) - DNS Management for Traefik

## Overview

Cloudflare Companion automatically manages DNS entries in Cloudflare for Traefik services.

## Setup

1. Create Cloudflare API token
2. Configure companion
3. Start companion
4. DNS entries are automatically created

## Dependencies

- Traefik must be running
- Cloudflare account with API token
